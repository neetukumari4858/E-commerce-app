# Royal 
Royal is an E-commerce website project built using REACT. It has features like filters, products page, wishlist page and cart page.
# Tech Stack
     React
     React Router
     Mockbee
     Font Awesome
# Features
- Home Page
    - Featured category
    - Redirect to the selected category
- Product Listing Page
    - Filter by Price
    - Filter by Category
    - Filter by Rating
    - Sort by price
    - Clear Filter
- Cart Management
    - Add to Cart
    - Remove From Cart
    - Move to Wishlist
    - Increase/decrease quantity of a item
- Wishlist Management
    - Add to Wishlist
    - Remove from Wishlist
    - Move to Cart
- Authentication
    - Sign-up 
    - Login
